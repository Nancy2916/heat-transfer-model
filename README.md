# heat-transfer-model

This project analyzes heat transfer performance in a heat exchanger using experimental data.

The data was processed using Python to calculate key parameters such as heat transfer rate (Q), Log Mean Temperature Difference (LMTD), and the overall heat transfer coefficient (U). The model compares parallel flow and counter flow heat exchanger modes.

The results are visualized using Python plots to understand how the overall heat transfer coefficient changes with flow rate.

Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

Key Concepts

- Heat Transfer Analysis
- LMTD Method
- Overall Heat Transfer Coefficient (U)
- Parallel vs Counter Flow Heat Exchangers

This project demonstrates how data analysis and basic programming can be used to study heat exchanger performance.
